# PHP_Text_Include
PHPで動く画像の右下に文字を入れるツール
PHP 5.4≦

JPGおよびPNGに対応

次のコードを入れることで画像に文字を入れることができます。
<code>/usr/local/bin/php example.jpg</code>

フォントは別途ご用意ください。
